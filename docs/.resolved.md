When navigating to the exercise view (BassTrainer), the application crashes with:

```
TypeError: Cannot read properties of null (reading 'useMemoCache')
    at BassTrainer (http://localhost:5173/src/BassTrainer.jsx:25:3)
```

### ğŸ” Root Cause

1. The project had `babel-plugin-react-compiler` enabled in `vite.config.js`
2. This compiler transforms React code to use internal optimization hooks like `useMemoCache`
3. The compiled code was cached in the browser
4. After disabling the compiler, the browser still serves the old compiled code

### âœ… Solution Applied

**File:** `vite.config.js`

```diff
  plugins: [
-   react({
-     babel: {
-       plugins: [['babel-plugin-react-compiler']],
-     },
-   }),
+   react(),
    tailwindcss(),
```
